# 个人成长型博客 PRD

## 一、产品目标
用于记录个人学习、成长与项目经历，作为长期博客与求职作品展示。

## 二、用户角色
- 普通用户（自己）
- 游客（未登录）

## 三、核心功能

### 1. 用户功能
- 注册：用户名、邮箱、密码
- 登录：账号密码登录
- 个人信息：查看 / 修改头像、简介

### 2. 文章功能
- 写文章：标题、正文、分类、标签、封面
- 编辑文章
- 删除文章
- 草稿 / 发布状态
- 查看文章详情
- 浏览次数统计

### 3. 分类与标签
- 文章分类
- 文章标签
- 按分类 / 标签筛选文章

### 4. 评论功能
- 发表评论
- 查看评论列表

## 四、页面设计

- 首页：文章列表
- 文章详情页
- 写文章页（需登录）
- 编辑文章页（需登录）
- 个人中心页
- 登录 / 注册页
- 关于我页面

## 五、非功能需求
- 支持分页
- 基础 SEO
- 响应式布局（PC 优先）

## 六、阶段说明
- 第一阶段：文章 + 用户（MVP）
- 第二阶段：评论、标签、优化体验
